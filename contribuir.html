Mejora el código fuente

El código fuente de este proyecto se encuentra en github, puedes modificar el código fuente y aportar de vuelta al proyecto a través de un Pull Request.

Ayúdanos a ayudarte

Siempre estamos buscando como mejorar el proyecto, añadir nuevas funciones, tutoriales, etc. cada vez que mejoramos trascendentAR, mejoramos las aplicaciones que tu creas y facilitamos tu experiencia. Si la programación de la librería no te interesa pero aún así quieres contribuir, considera hacer una donación al proyecto, así crearemos una herramienta mejor para que las aplicaciones que tu hagas sean mucho más increíbles.
